What exactly is a favicon?

A favicon (short for favorite icon) also known as a shortcut icon, website icon, tab icon, URL icon, or bookmark icon, is a file containing one or more small icons, associated with a particular website or web page.

How are favicons created?

A web designer can create such an icon and upload it to a webite (or web page) by several means, and grpahical web browsers will then make use of it.Browsers that provide favicon support typically display a page's favicon in the browser's address bar.

How did I create my favicon?

I created an account on StudioWeb.org. Then, I watched tutorials on how to make favicons. Last, I was able to achieve making a 32 x 32 pixel favicon.
